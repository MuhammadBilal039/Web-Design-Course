HTML Tutorial

HTML stands for Hyper Text Markup Language

HTML is the standard markup language for creating Web pages.

HTML describes the structure of Web pages using markup.

HTML elements are the building blocks of HTML pages.

HTML elements are represented by tags.

HTML elements tell the browser how to display the content.

Tags of HTML

HTML tags are keywords surrounded by angle brackets.

HTML tags normally come in pairs like <h1> and </h1>.

HTML tags also tell the browser how to display the content.

HTML tags are also known as elements.

HTML elements tell the browser how to display the content.

HTML Elements

HTML elements are represented by tags.

HTML elements tell the browser how to display the content.

HTML elements are the building blocks of HTML pages.

Some HTML elements are following below:

Head Element

The head element contains meta information about the document.

Body Element

The body element contains the visible page content.

Title Element

The title element defines the title of the document.

Heading Elements

The heading elements (h1, h2, h3, h4, h 5, h6) defines headings for your content.

Paragraph Element

The paragraph element (p) defines a paragraph.

Div Element

The div element defines a division or a section in an HTML document.

Span Element

The span element defines a section in a document.

Unordered List

The unordered list element defines an unordered list of items.

Ordered List

The ordered list element defines an ordered list of items.

Definition List

The definition list element defines a definition list.

Lists Element

The list element defines a list of items.

Link Element

The link element defines the relationship between a document and an external resource.

Image Element

The image element defines an image in an HTML document.
